# WPF Vector Viewer

A simple vector viewer built in WPF and C# for a coding challenge

## Screenshot
![demo-gif](https://github.com/user-attachments/assets/5b5fedca-fed1-4a30-9992-79dbce181412)

## Features
- Json format supported
- Zoom percentage
- Easy to extend

## Technologies

- .NET 8.0
- C#
- WPF

## Getting Started

1. Clone the repository
2. Open `WSCAD.sln` in Visual Studio
3. Build and run
4. Click open Json file, select sample given in src/Resources/Sample.json or any supported json file
